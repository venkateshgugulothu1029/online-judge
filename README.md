# Online Code Judge 🚀

## 📌 Overview
This is a simple online coding platform where users can submit code and get results based on predefined test cases.

## ⚙️ Features
- Code submission API
- Automatic code execution
- Output validation with test cases
- Fast and simple backend system

## 🛠 Tech Stack
- Python
- Flask

## ▶️ How to Run

1. Install dependencies:
pip install flask

2. Run the server:
python app.py

3. Open in browser:
http://127.0.0.1:5000/

## 📡 API Example

POST /submit

  "code": "print('0 1')"
